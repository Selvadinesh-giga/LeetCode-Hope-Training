
#  Online Shopping Cart System (Java OOP Project)

##  Project Overview
**Project Name:** Online Shopping Cart System  
**Language:** Java  
**Version:** 1.0  
**Date:** April 2026  

This is a command-line based e-commerce application developed in Java to demonstrate Object-Oriented Programming (OOP) concepts.

##  Key Features
- User Management
- Product Management
- Shopping Cart
- Discount System
- Search Functionality
- Menu-driven UI

##  OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

##  Project Structure
- Product.java
- PhysicalProduct.java
- DigitalProduct.java
- User.java
- CartItem.java
- ShoppingCart.java
- Discountable.java
- CouponDiscount.java
- ProductCatalog.java
- ShoppingCartApp.java

##  How to Run
Compile:
javac -encoding UTF-8 *.java

Run:
java ShoppingCartApp
